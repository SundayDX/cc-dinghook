# 更新日志

所有重要的项目变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2024-01-13

### 新增
- 🎉 首次发布 CC-DingHook
- ✨ 支持全局 Claude Code hook 集成
- 📱 钉钉通知功能
- 🔧 灵活的配置系统
- 🔒 支持钉钉安全签名
- 📊 执行时长和状态统计
- 🎯 智能通知过滤（成功/失败/错误）
- 📝 美观的 Markdown 消息格式

### 功能特性
- 🚀 一键安装脚本
- 🔧 命令行配置工具
- 📋 详细的配置选项
- 🛡️ 安全的配置管理
- 🔍 完整的错误处理
- 📖 详细的文档说明

### 支持的安装方式
- 🌐 在线一键安装
- 📦 手动下载安装
- 🔧 Git 克隆安装
- 🐍 Python 单文件运行

### 配置选项
- Webhook URL 配置
- 安全密钥支持
- 消息模板自定义
- 通知条件控制
- 执行信息包含选项
